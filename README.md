# ISTQB Foundation 문제은행 V1.0 - 모의고사 모드 연결

ISTQB Foundation 샘플문제 4종 전체 186문항에 초보자 튜터형 해설을 적용한 V1.0 1차 완성본입니다. 이번 버전에서는 `학습 모드 / 모의고사 모드`를 분리하고, 모의고사 모드에 40문항, 60분 타이머, 풀이 중 해설 비공개, 제출 후 채점 및 합격 판정 로직을 연결했습니다.

## 실행

```bash
python -m http.server 8000
```

브라우저에서 접속:

```text
http://localhost:8000
```

## V1.0 Full - Question Layout Fix

V1.7A의 일부 문항은 PDF 원문에서 표, 목록, 결함 보고서 형태로 구성되어 있었으나, JSON 텍스트 추출 과정에서 줄바꿈과 표 구조가 사라져 모바일 화면에서 가독성이 낮아질 수 있었습니다.

이번 수정에서는 원본 `question` 텍스트는 그대로 보존하고, 아래 5개 문항에 `questionBlocks` 필드를 추가했습니다.

- V1.7A 21번: 성적 구간 목록 + 테스트 케이스 표
- V1.7A 22번: 기능 목록 + 결정 테이블
- V1.7A 33번: 테스트 케이스 우선순위/종속성 표
- V1.7A 35번: 리스크 정보 목록 + 대응 방법 하위 목록
- V1.7A 38번: 결함 보고서 카드형 표시

앱 로직은 `questionBlocks`가 있는 문항은 구조화된 HTML로 표시하고, 없는 문항은 기존처럼 `question` 텍스트를 표시합니다.

## 학습 모드 / 모의고사 모드

선택 화면에서 두 가지 학습 방식을 선택할 수 있습니다.

- 학습 모드: 즉시 채점, 초보자 튜터형 해설, 원문 해설, 오답노트, 북마크 기능을 사용합니다.
- 모의고사 모드: 본시험 40문항만 사용하고, 제한시간 60분 동안 풀이합니다. 풀이 중에는 정답과 해설이 표시되지 않으며, 제출 후 26점 이상 합격 기준으로 채점합니다.

모의고사 모드 추가 기능:

- 고정 샘플지 모의고사: V1.7A / V1.7B / V1.6 / V1.5 각각 본시험 40문항
- 랜덤 실전 모의고사: 전체 본시험 160문항 중 40문항 무작위 추출
- 이전/다음 이동
- 문제 번호 네비게이션
- 검토 표시
- 시간 종료 시 자동 제출
- 제출 후 오답 해설 보기 / 전체 해설 보기
