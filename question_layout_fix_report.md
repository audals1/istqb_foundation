## 수정 내용

### 5번

- 원본 PDF의 i~v 항목처럼 각 항목을 별도 줄로 표시하도록 변경했다.
- 기존 카드형/목록형보다 7번 문항의 원문 줄바꿈 느낌에 가깝도록 `sourceLines` 표시를 적용했다.

### 13번

- 장애 유형 1~4를 각각 별도 줄로 표시하도록 변경했다.
- 테스트 레벨 A~D도 각각 별도 줄로 표시하도록 유지했다.
- 표 형태가 아니라 원본 PDF와 비슷한 세로 나열형으로 변경했다.

### 14번

- 원본 PDF 배치에 맞춰 별도 수정했다.
- AC1/AC2/AC3 설명을 불필요한 목록형으로 풀지 않고 원문 흐름처럼 문단으로 표시했다.
- 테스트 실행 내역은 표로 유지하되 첫 번째 열을 TC1/TC2/TC3 라벨 열로 정리했다.

### 17번

- `•` 항목을 각각 별도 줄로 표시했다.
- bullet 전 문장과 bullet 후 질문 문장을 별도 문단으로 분리했다.

### 20번

- `• 층`, `• 정원` 항목을 각각 별도 줄로 표시했다.
- bullet 전 문장과 후속 설명 문장을 별도 문단으로 분리했다.

### 32번

- 3점 추정값 3개를 각각 별도 bullet 줄로 표시했다.
- bullet 전 문장과 마지막 질문 문장을 별도 문단으로 분리했다.

## 구현 방식

- 원본 `question` 텍스트는 변경하지 않았다.
- 화면 표시용 `questionBlocks`만 수정했다.
- `sourceLines`, `sourceBullets` CSS 클래스를 추가해 원문형 줄바꿈과 bullet 가독성을 개선했다.
- V1.7A Beta 2 JSON과 통합 Beta 2 JSON에 동일하게 반영했다.

## 변경하지 않은 것

- 정답
- 보기
- 원문 해설 `explanation`
- 튜터형 학습 해설 `learningExplanation`
- LO / K-Level / 배점
