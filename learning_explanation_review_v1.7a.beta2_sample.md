# V1.0 Beta 2 - V1.7A 초보자 튜터형 해설 샘플 검수 리포트

## 목적

기존 V1.0 Beta 해설이 원문 해설을 보기 좋게 재정렬한 수준에 가까웠기 때문에, V1.7A 1~10번만 먼저 초보자 튜터형 해설로 재작성했습니다.

## 적용 범위

- 대상: V1.7A 1~10번
- 데이터 파일: `data/istqb-v1.7a.v1.0-beta2.json`, `data/all.v1.0-beta2.json`
- 원본 보존: `data/all.json`, `data/all.v1.0-beta.json`, `data/istqb-v1.7a.json`, `data/istqb-v1.7a.v1.0-beta.json`

## 해설 작성 기준

각 문항은 아래 항목을 포함하도록 작성했습니다.

1. 문제 의도: 이 문제가 무엇을 묻는지
2. 배경 개념: 초보자가 먼저 알아야 할 ISTQB 개념
3. 풀이 흐름: 어떤 순서로 판단해야 하는지
4. 정답 상세 이유: 왜 이 선택지가 정답인지
5. 오답 상세 이유: 그럴듯한 오답이 왜 틀렸는지
6. 초보자 함정: 시험에서 헷갈리는 지점
7. 다음 풀이 기준: 유사 문제를 다시 만났을 때 판단법
8. 한 줄 암기 포인트
